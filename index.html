Sportcred is a sports fanatic's dream web application

- including a live chat, current scores and results from NBA games, a scoring system used to track sports knowledge, a debates and analysis section used to compare and share your thoughts with others, and other sport-related features.

This was a full stack development project, done in a team of 7 members over a period of 2 months using an Agile development process.

The following technologies were used; Jira, ReactJS, Java, Neo4j, RESTful services.

Click here to learn more and check out the github page!


This chat server is a multi-user real-time chat platform developed in C.

This server uses custom runtime arguments to determine which port to run on, and I/O multiplexing to handle multiple simultaneous requests.

The server uses EPOLL functions in order to handle available clients, and continuously process data as well as socket connections to connect/handle clients.


JavaMockShell is a (mock) shell with over 18 terminal commands, incorporating redirection and a mock file system for files to be created, saved, and displayed.

This project implemented numerous design patterns including Singleton, Factory Methods, and Dependency Injection.

This was developed solely in Java, and a team of 4 members using a strict Agile scrum-development process.

This project also includes a set of exhaustive JUnit tests for each command to ensure correctness.
